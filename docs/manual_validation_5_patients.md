# Ручная валидация медицинских расчетов: 5 контрольных пациентов

Этот файл нужен для ручной сверки результатов кода с внешними онлайн-калькуляторами.

## Важно

Ожидаемые значения ниже рассчитаны по текущей логике проекта:

- CKD-EPI 2021 creatinine;
- Cockcroft–Gault;
- ACR в мг/ммоль;
- пересчет ACR в мг/г через `× 8.84`;
- стадия ХБП в формате проекта: `С1`, `С2`, `С3а`, `С3б`, `С4`, `С5`;
- прогноз KDIGO по сочетанию стадии СКФ и категории альбуминурии.

Эти значения нужно использовать как чек-лист:
1. запустить наш код;
2. получить результат;
3. ввести эти же данные во внешние калькуляторы;
4. зафиксировать расхождения, если они есть.

## Пациент 5 — защитный сценарий: врач случайно ввел креатинин крови не в тех единицах

Этот сценарий нужен не как реальный пациент, а как проверка слабого места системы.

Смысл: если в поле “креатинин крови, мкмоль/л” врач случайно введет значение из другой системы единиц, например `1.2` как будто это mg/dL, код не сможет понять ошибку и посчитает так, будто это `1.2 мкмоль/л`.

### Входные данные

| Параметр | Значение |
|---|---:|
| Пол | Женский |
| Дата рождения | 1986-01-01 |
| Дата приема | 2026-01-01 |
| Возраст для расчета | 40 лет |
| Масса тела | 60 кг |
| Креатинин крови | 1.2 мкмоль/л |
| Альбумин мочи | 0.03 г/л |
| Креатинин мочи | 10 ммоль/л |

### Пересчет единиц для ACR

```text
0.03 г/л = 30 мг/л
10 ммоль/л = 10 ммоль/л
ACR = 30 / 10 = 3 мг/ммоль
```

### Вывод по пациенту 5

Это ожидаемое поведение текущего кода, но медицински это выглядит как явная ошибка исходных данных.

После ручной проверки стоит добавить в интерфейс/бэкенд предупреждение по правдоподобности:


## Проверка взаимопересчитываемых значений ACR

### Пациент 2 и пациент 5

Оба должны дать одинаковый ACR:

```text
30 мг/л / 10 ммоль/л = 3 мг/ммоль
0.03 г/л / 10 ммоль/л = 3 мг/ммоль
```

### Пациент 3

```text
0.31 г/л / 10000 мкмоль/л
= 310 мг/л / 10 ммоль/л
= 31 мг/ммоль
```

### Пациент 4

```text
300 мг/л / 10000 мкмоль/л
= 300 мг/л / 10 ммоль/л
= 30 мг/ммоль
```

## Что сверять во внешних калькуляторах

### CKD-EPI 2021

Вводить:

```text
пол
возраст
креатинин крови
единица креатинина: мкмоль/л
формула: CKD-EPI 2021 creatinine
```

### Cockcroft–Gault

Вводить:

```text
пол
возраст
масса тела
креатинин крови
единица креатинина: мкмоль/л или mg/dL
```

Важно: наш код использует фактическую массу тела, без ideal/adjusted body weight.

### ACR

Вводить:

```text
альбумин мочи
единица альбумина
креатинин мочи
единица креатинина мочи
```

Если калькулятор не поддерживает нужную единицу, пересчитать вручную:

```text
г/л -> мг/л: умножить на 1000
мкмоль/л -> ммоль/л: разделить на 1000
```
