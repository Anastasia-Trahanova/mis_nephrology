# app/db

Папка `app/db` — это технический слой подключения приложения к PostgreSQL.

## Что здесь лежит

- `connection.py` — настройки DSN, пул подключений `ThreadedConnectionPool`, обёртка `PooledConnection` и функция `get_db_connection()`.
- `__init__.py` — короткий публичный импорт для подключения к БД.

## Когда редактировать

Редактируй эту папку, если нужно поменять:

- способ подключения к PostgreSQL;
- размер пула подключений;
- поведение commit/rollback;
- техническую обёртку соединения.

## Когда не редактировать

Не добавляй сюда SQL-запросы по пациентам, приёмам, анализам, диагнозам или карточке пациента. Для этого есть `app/repositories` и `app/services`.

## Важное правило

`app/db` ничего не знает про медицинскую логику. Это только подключение к базе.
